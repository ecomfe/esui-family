ESUI Family

    初始化项目

    在当前目录下，通过命令edp project init可以将当前目录初始化成一个项目目录。

    
myProject/
      ├── .edpproj   
      ├── dep      
      ├── doc      
      ├── src
      ├── test
      ├── tool
      ├── edp-build-config.js
      ├── edp-webserver-config.js
      ├── module.conf
      ├── package.json
      └── README

    配置文件

    在项目路径下创建assets文件夹，并在该文件夹下创建配置文件config.js。

    
require.config({
    'packages': [],
    'paths': {}
});

    导入依赖包

    在项目路径下执行命令edp import esui和命令edp import ub-ria-ui导入所有依赖包。

    导入的包将存放在项目的dep目录下面。

    
dep/
  ├── eicons   
  ├── eoo      
  ├── esf      
  ├── est
  ├── esui
  ├── etpl
  ├── jquery
  ├── moment
  ├── ub-ria-ui
  └── unserscore

    加载组件样式

    在src/css路径下创建main.less文件，在其中加载组件样式。

    在该路径下执行编译命令lessc main.less main.css生成main.css文件。

    
// import est
@import "../../dep/est/1.3.0/src/all.less";
.global-normalize();

// import esf
@import "../../dep/esf/1.0.0-beta.2/theme/standard/main.less";

// import esui
@import "../../dep/esui/3.2.0-beta.1/theme/standard/Main.less";

// import ub-ria-ui
@import "../../dep/ub-ria-ui/1.0.0-beta.1/theme/standard/Main.less";

// import eicons
@import "../../dep/eicons/1.0.0-beta.1/src/main.less";

// font
@fa-font-path: "../../dep/eicons/1.0.0-beta.1/fonts/font-awesome";

// resource
@advanced-color-resource-path: "../../dep/ub-ria-ui/1.0.0-beta.1/resource";

// loaders are mixins now, output them manually
// 下面任选其一即可
.esui-loader-bouncers();
.esui-loader-border();
.esui-loader-borders();
.esui-loader-spinner-wave();
.esui-loader-rotating-plane();
.esui-loader-chasing-dots();
.esui-loader-spinner-circles();

    HTML页面代码

    在项目路径下创建文件index.html,在页面中载入样式文件和脚本文件。

    样式文件：src/main.css

    脚本文件：http://s1.bdstatic.com/r/www/cache/ecom/esl/2-0-6/esl.js和assets/config.js

    载入文件后，在页面上创建script标签，require需要使用的组件并进行初始化。下面给出button控件的示例：

    
<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8" />
        <title></title>
        <link rel="stylesheet" href="src/main.css" />
    </head>
    <body>
        <button data-ui="type:Button;variants:primary">Primary</button>
        <script src="http://s1.bdstatic.com/r/www/cache/ecom/esl/2-0-6/esl.js"></script>
        <script src="assets/config.js"></script>
        <script>
            require(
                [
                    'esui',
                    'esui/Button'
                ],
                function (ui) {
                    ui.init();
                }
            );
        </script>
    </body>
</html>

    启动项目

    在项目路径下执行命令edp webserver start启动调试。
